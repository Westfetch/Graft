Origin

"Built by someone so allergic to HMRC that they automated their entire working life instead."

It started with procrastination. Couldn't face logging into HMRC, so built ResellerHQ instead. Then Lifemap, to see where all those delivery miles actually went. Then TaxFlow was conceived, because every product kept generating data with nowhere to go.

Then The Pipe, because delivery drivers were leaving money on the table every shift. Then VanHQ, because the removals industry had no decent dispatch infrastructure. Then ExtraAgent, because actors were doing the same admin manually that everything else was automating.

Every product in this ecosystem exists because someone with a real job in the gig economy needed it and couldn't find it. That's the only filter that matters here.
